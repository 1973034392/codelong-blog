## 组件概述

服务发送SDK是为Java应用设计的轻量级HTTP客户端库，提供简单API调用网关服务并处理认证、重试、异常等逻辑。核心功能：

- 自动生成JWT令牌
- 内置连接池管理
- 支持GET/POST/PUT/DELETE方法
- 统一异常处理
- 响应结果自动解析

## 核心类说明

|  `HttpClientFactory`   | 创建带连接池的HTTP客户端 |

| `GatewaySDKProperties` |        配置参数类        |
|   `GatewayException`   |     统一异常处理基类     |
